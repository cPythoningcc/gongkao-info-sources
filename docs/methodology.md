# 信息核查方法

## 1. 线索发现

可使用考公机构、培训机构、招聘聚合网站快速发现公告线索，例如：

- 华图、中公、粉笔、格木教育等机构站
- 事业单位招聘考试网、上岸鸭、公考日历等聚合站
- 本地宝、地方招聘汇总页等本地信息站

这类网站适合发现线索，但不作为最终依据。

## 2. 官方核验

对每条线索，应尽量回到以下一手来源核验：

- 国家公务员局专题网站
- 省/市/县组织部、公务员局
- 省/市/县人社厅/人社局
- 省/市/县人事考试网
- 政府门户网站的招聘公告、人事信息、招考信息栏目
- 招聘单位官网或主管部门官网
- 官方报名系统

## 3. 岗位表核验

优先下载官方附件：

- 公告原文
- 岗位表 / 职位表
- 专业参考目录
- 报名指南
- 联系方式表

核验时重点看：

- 报名起止时间
- 招聘对象
- 学历学位
- 专业名称与专业代码
- 政治面貌
- 户籍/生源
- 年龄
- 工作经历
- 职称/证书/资格要求
- 是否编制内、编外、合同制、劳务派遣

## 4. 状态标注

建议状态字段：

- `已核验官方`
- `已核验官方并下载岗位表`
- `官方待核验`
- `官方未核验`
- `过程类信息/非新报名`
- `非有效招考公告`

报名状态字段：

- `未截止`
- `已截止`
- `待核验`
- `非报名线索`
- `待发布/需关注`

## 5. 注意事项

- 机构网站标题可能存在地区误标、旧公告混入、预测信息混入等问题。
- 有些官方公告可能发布在微信公众号，网页搜索不一定能直接检索到。
- 有些附件下载需要浏览器、Referer、验证码或报名系统登录。
- 最终是否能报，以官方报名系统资格审查为准。
